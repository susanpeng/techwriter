8.4. Exporting to XSL

To export your bookshelf file to XSL:

  1. In the menu bar, click File > Export Bookshelf.
  2. Select XSL Document, specify the name for XSL file, and click Export Now.

    Bookshelf Editor creates an XSL file with the name you specified in the Export Bookshelf dialog box in the directory where you installed Bookshelf Editor 1.1. The name of the export directory appears in the Directory for Export field.
