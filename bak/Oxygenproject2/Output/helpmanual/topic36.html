4.5. Saving file with a different name

To save bookshelf file with a different name:

  1. From the File menu, click Save As.
  2. In the Save As dialog box, type the new file name in the File name entry field and click Save.

    The bookshelf file is saved with the new file name to the bookshelfeditor directory.
