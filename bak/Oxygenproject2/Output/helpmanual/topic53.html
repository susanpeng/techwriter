7.3. Changing the size of the space

To change the size of the space:

  1. Select the space you want to change.
  2. Do one of the following:
    • In the Tool Panel, click Book Properties.
    • On the Object menu, click Book Properties.
  3. In the Space Properties dialog box, move the slider to a desired space size.
  4. Click OK.

    The size of the space is changed to the specified size.
