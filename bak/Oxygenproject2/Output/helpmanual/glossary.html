Glossary

book
a graphic image of a book used for storing URLs for later retrieval in a web browser.
GUI
Graphical User Interface. The common method of interacting with a computer that allows any graphics image to be displayed on screen.
HTML
HyperText Markup Language. It is the language of web pagesâ€”the markup language that browsers read to render web pages.
JPEG
Joint Photographic Expert Group and it is used as a method for compressing digital images.
URL
Uniform Resource Locator. A URL is a formatted text string used by Web browsers to identify a network resource on the Internet.
shelf
a graphics container for storing books.
XML
Extensible Markup Language. XML is an open standard for describing data from the W3C. It is used for defining data elements on a Web page.
XSL
Extensible Stylesheet Language. XSL is a style sheet standard from the W3C that is used to convert an XML document into many formats for publishing and printing.
Breadcrumbs
Appear horizontally across the top of a web page, provide a trail for the user to follow back to the starting or entry point.
