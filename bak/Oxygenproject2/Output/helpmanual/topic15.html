3.2.3. Bookshelf Menu

The following shows the open Bookshelf menu and explains its features.

The Bookshelf menu consists of the following Bookshelf Editor commands:

  • Design Properties

    Opens the Design Properties dialog box. For details, see Design Properties.

  • Shelf Properties

    Opens the Shelf Properties dialog box. For details, see Shelf Properties.

  • Add New Shelf

    Opens the Shelf Properties dialog box. For details, see Shelf Properties.

  • Delete Selected Shelf

    Deletes the selected shelf.

  • ML Viewer

    Allows you to view your bookshelf either in XML or HTML. For details, see ML Viewer.
