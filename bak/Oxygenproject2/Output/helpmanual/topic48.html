6.6. Adding a URL to the book

To add a URL to a book:

  1. Select a book.
  2. Do one of the following:
    • In the Tool Panel, click Book Properties.
    • On the Object menu, click Book Properties.
  3. In the Object Properties dialog box, in the URL Link field, enter a web address, and then click OK.

    The URL link you have entered appears in the Object Link field of the Tool Panel.

Note: The web address you enter in the URL Link field must start with "http://" or "https://"and should not have white spaces in it.
