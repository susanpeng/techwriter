Installation Instructions
for VLC Media Player


VLC media player is an application that lets you easily play music, watch videos, and browse photos on your computer. These instructions will show you how to download it from www.cnet.com website and install it on your Mac.

Find VLC media player

  1. Open the browser you prefer to use.
  2. Type www.cnet.com in the address bar to open the website.
  3. On the menu bar, hover your mouse over Download. You will see five items in the drop-down menu.

  4. Click Mac Software. You will see a page with VLC Media Player on its right side.

Download VCL media player

  1. Click VLC Media Player. You will see the download page for VLC media player.

  2. Click . Downloading starts.

You can monitor the download progress with the indicator at the bottom left corner of the screen.
Start In progress Done

Install VLC media player

  1. Click the down arrow next to the file. A pop-up menu displays.

  2. Click Show in Finder. A finder window opens with downloaded file named vlc-2.1.0.dmg inside.
  3. Double click the file named vlc-.2.1.0.dmg. The VLC media player starts installing. When installation is finished, a start window displays.

  4. Double click VLC icon. A dialogue box displays to ask if you want to open the VLC media player.

  5. Click Open button. The VLC Media Player will start.

Congratulations! You have successfully downloaded and installed the VLC media player on your Mac.
Now, you are able to use VLC media player to play music, watch videos, and browse photos.
