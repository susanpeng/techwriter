Welcome to the Quick Start for Aegisub.

Aegisub is a software program for adding subtitles to videos. This Quick Start Manual is to help you get basic tasks done in a minimal time.

This online help includes the following topics:

  1. About Aegisub
  2. Main Screen
  3. Editing Subtitles
  4. Working with styles
  5. The Styles Manager
  6. The Style Editor
  7. Designing a Style
  8. Timing Subtitles to Audio

Even though only a few topics are included in this Quick Start, they are enough for you to get started with Aegisub and start adding subtitles to a video. After you get familiar with fundamental operations, you can find advanced topics in the complete manual from Aegisub’s website later.
