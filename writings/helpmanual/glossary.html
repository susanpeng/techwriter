Glossary

book
A graphic image of a book used for storing a URL for later retrieval in a web browser.
GUI
Graphical User Interface. The common method of interacting with a computer that allows any graphic image to be displayed on the screen.
HTML
HyperText Markup Language. The language of web pagesâ€”the markup language that browsers read to render web pages.
JPEG
Joint Photographic Expert Group. It is a standard image format for compressing digital images.
URL
Uniform Resource Locator. A URL is a formatted text string used by web browsers to identify a network resource on the Internet.
shelf
A graphic container for storing books.
XML
Extensible Markup Language. An open standard for describing data from the W3C. It is a powerful way to store data in a format that can be stored, searched, and shared.
XSL
Extensible Stylesheet Language. A style sheet standard from the W3C that is used to transform and present an XML document into many formats for publishing and printing.
breadcrumbs
Appear horizontally across the top of a web page, provide a trail for the user to follow back to the starting or entry point.
