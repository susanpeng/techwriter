2.2. Installing the software

To install Bookshelf Editor 1.1:

  1. In the Downloads directory, locate BookshelfEditor11.exe and double-click BookshelfEditor11.exe.
  2. In the Security Warning dialog box, click Run.
  3. In the Bookshelf Editor 1.1 Install dialog box, click Next.
  4. In the next dialog box that opens, select I have read this ... and click Next.
  5. In the next dialog box that opens, click OK.

    Bookshelf Editor 1.1 is now installed on your computer.
