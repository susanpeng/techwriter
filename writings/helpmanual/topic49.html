6.7. Viewing book properties

To view properties of the book:

  1. Select a book.
  2. Do one of the following:
    • In the Tool Panel, click Book Properties.
    • On the Object menu, click Book Properties.
  3. In the Object Properties dialog box, view the following object properties:
    • Style
    • Colour
    • Title
    • URL Link
    • Object Width
    • Building Width
