3.2.4. Object Menu

The following shows the open Object menu and explains its features.

The Object menu consists of the following Bookshelf Editor commands:

  • Insert

    Allows you to add Narrow (Default) book, Wide Book, or Space to the selected shelf.

  • Book Properties

    Opens either the Object Properties dialog box or the Space Properties dialog box, where you can modify the properties of the selected book or space, respectively.

  • Move Left

    Moves the selected book or space one position to the left on the shelf.

  • Move Right

    Moves the selected book or space one position to the right on the shelf.

  • Delete

    Removes the selected book or space from the shelf.
