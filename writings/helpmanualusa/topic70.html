6. Recommendations

The feedback from testers was mainly positive about the quality and helpfulness of the documentation content. 

Here are recommendations for changes required to improve the quality and usability of the User's Manual.

  1. Add numbering for the sections and sub-sections of the table of contents on the left side of the pages.
  2. Renam the Narrow (Default) book to "narrow (default) book".
  3. Rework awkwardly worded sentences like "Select the book to which you want to apply a color change".
  4. Add missing terms to the index (HTML, title, and exporting).
  5. Add "Breadcrumbs" to the glossary, and update its description in section  1.3. How to use this manua .
  6. Section 8.2. Exporting to HTML can also be improved by adding a step to clarify the context. 
  7. The 3.3.5. ML Viewer section could be re-labelled so that users understand that it is for viewing XML and HTML code. 
  8. The section on viewing HTML should have an example showing where the HTML file is most likely to be found. 
  9. Learning the Graphical User Interface (GUI) and Downloading and Installing the Bookshelf Editor sections should switch places.
  10. Remove the line "The new Bookshelf confirmation ..." below the screenshot on step 1 in section 4.1. Starting a new file.
  11. Add the alternative method and remove the note in section page 4.3.1. Adding a Bookshelf Title.
  12. Add the screenshots for the alternative menu bar method in sub-sections: 4.3.2. View the Construction, 4.3.3. View the Background, and 4.3.4. View the Statistics.
  13. Delete the duplicate line in section 5.2. Deleting a shelf, step1.
  14. Add a screenshot for the alternative method and update the screenshot to show the two shelves in section 5.3. Changing the size of the shelf.
  15. Restructure the steps in section 9.1. Opening the Bookshelf in your browser.
  16. Restructure the steps, add description of the cursor changes to a hand icon when the mouse hovers over a book link, and change "select" to "click" in section 9.2. Accessing book contents.
