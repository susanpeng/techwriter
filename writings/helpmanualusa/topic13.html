3.2.1. File Menu

The following shows the open File menu and explains its features.

The File menu consists of the following Bookshelf Editor commands:

  • New Bookshelf

    Creates a new bookshelf file.

  • Open

    Opens an existing bookshelf file.

  • Save

    Saves the current bookshelf file.

  • Save As

    Save the current bookshelf file with a different name.

  • Export Bookshelf

    Export the current bookshelf project to various file formats.

  • Exit

    Exits Bookshelf Editor.
