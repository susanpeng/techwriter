10. Hot-Key

All hot-keys are described in the folowing table:

Hot-KeyCommandFunction
Ctrl-NNew Bookshelf...Create a new bookshelf file
Ctrl-OOpen...Open an existing bookshelf file
Ctrl-SSaveSave current bookshelf file
Crtl-JBook Properties...Open Object Properties dialog box
Crtl-LeftMove LeftMove selected book left
Ctrl-RightMove RightMove selected book right
Ctrl-DeleteDeleteDelete selected book
