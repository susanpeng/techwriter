6.1. Adding a book

To add a book to a shelf:

  1. Select the shelf where you want to add a book.

    The red arrow indicates the currently selected shelf.

  2. Do one of the following:
    • Under the Tool Panel, in the Object Palette, click the icon of a book you want to add.
    • On the Object menu, point to Insert, and then click Narrow (Default) book or Wide Book.

    The book appears aligned to the left.
